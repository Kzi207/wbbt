# Web Bot Manager

Giao dien web JSX de quan ly bot.

Phien ban nay su dung package cua project chinh (khong can `package.json` rieng trong `web`).

## Tinh nang

- Thay cookie khi cookie cu bi loi
- Quan ly nhom bot dang tham gia
- Quan ly thue bot:
  - Them nhom thue bot theo UID nhom
  - Tao key thue bot
  - Xoa nhom thue bot
- Quan ly bat/tat lenh theo nhom
- Quan ly ban/unban user va nhom

## Chay local

```bash
cd web
node server/index.cjs
```

Mac dinh:

- Frontend + API: http://localhost:7070

Luu y: can dam bao da cai dependency o thu muc goc project.
